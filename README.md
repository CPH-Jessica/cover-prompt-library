## How to use this repo

Copy any prompt you need, paste it into Gemini, Grok, Ideogram, Midjourney, or whichever AI image tool you use. Triple-click a prompt to select the whole thing. Adjust the focal subject, color palette, or mood to match your specific book.

Then bring the image into Book Brush, Canva, Photoshop, or any design tool to do the typography, layout, and finishing work. That hybrid workflow is what makes the cover registrable as a copyrighted work (per the US Copyright Office's 2025 guidance — see the rules guide for details).

If you publish on KDP, check the AI-generated disclosure box. It's confidential. Failure to disclose is what gets books pulled, not the disclosure itself.
